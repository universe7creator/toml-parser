Frequently Asked

Everything you need to know

What is TOML?

TOML (Tom's Obvious, Minimal Language) is a configuration file format that's easy to read and write. It's used by Rust (Cargo), Python (pyproject.toml), and many other modern tools.

Is my data secure?

Absolutely. All parsing happens entirely in your browser using JavaScript. Your TOML files never leave your device or get sent to any server.

What TOML features are supported?

We support the full TOML 1.0 specification including tables, inline tables, arrays, strings (literal and basic), integers, floats, booleans, dates, and times.

Can I get a refund?

Do I need an internet connection?

After the initial page load, the parser works entirely offline in your browser. No internet connection required for conversions.
